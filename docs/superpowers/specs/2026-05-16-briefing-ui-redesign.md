# Briefing UI Redesign — 설계 문서

**날짜:** 2026-05-16  
**범위:** 브리핑 페이지 UI 두 영역 개선

---

## 개요

두 가지 UI 개선:

1. **우측 사이드바 개편** — 시장 지표 슬림화 + 조용한 CTA 추가 + 정확도 카드 시안성 개선
2. **아카이브 UI 교체** — 아코디언 → 카드 리스트

---

## 1. 우측 사이드바

### 1-1. 예측 정확도 카드 (Option C)

**현재 문제:** 22px 숫자 + 10px 레이블 + 9px 원형 점의 조합이 시각적으로 단조롭고 한눈에 파악하기 어려움.

**변경:**

```
[ 75%        ]  [ 68%        ]   ← 컬러 배경 타일 (7일: 초록, 30일: 노랑)
[ 최근 7일   ]  [ 최근 30일  ]

🔥 최근 3연속 적중                ← 스트릭 행 (연속 적중 시만 표시)

■ ■ □ ■ ■ ■ □ ■ ■ □            ← 10px 둥근 사각형 점 (현행 원형 → 사각형)
```

- **컬러 배경:** `acc-good` → 초록 타일(`#F0FDF4`), `acc-mid` → 노랑 타일(`#FEFCE8`), `acc-bad` → 빨강 타일(`#FEF2F2`)
- **스트릭:** `accuracy.recent_dots`에서 끝에서부터 연속 `correct=True` 개수 계산. N=1이면 표시 안 함, N≥2이면 `🔥 최근 N연속 적중` 표시
- **점 모양:** `border-radius: 3px` (원형 → 둥근 사각형)
- **데이터:** `generate_html.py`에서 `streak` 값 계산 후 템플릿에 전달

### 1-2. 시장 지표 — 브리핑 유형별 4개

**현재:** 브리핑 유형 무관하게 9개 지수 전부 표시.

**변경:** `briefing_type`에 따라 다른 지수 목록 전달.

| 브리핑 유형 | 표시 지수 |
|------------|----------|
| `kospi` (코스피 시초가/마감) | KOSPI, KOSDAQ, USD/KRW, 공포탐욕지수 |
| `us` (미국 시장) | NASDAQ, NQ, SOX, 공포탐욕지수 |

- `generate_html.py`의 `sidebar_items` 생성 로직에서 `briefing_type`으로 분기
- 공포탐욕지수(`type: 'fg'`)는 현재 `index.html`에만 있음 → `briefing.html`에도 추가

### 1-3. 조용한 CTA 블록

사이드바 최하단에 정적 HTML로 추가. 데이터 연동 불필요.

```html
<!-- 텔레그램 링크 (한 줄) -->
텔레그램 채널 →  (아이콘 + 링크)

<!-- 이메일 구독 -->
[이메일 입력창] [구독하기]
```

- 이메일 form은 기존 `/api/subscribe.mjs` POST 엔드포인트 사용
- 스타일: 배경 없음, 얇은 상단 구분선, 레이블 작게 (12px)

## 2. 아카이브 UI — 카드 리스트

### 현재 구조

`briefing-archive` 영역: 접힌 아코디언 행. 클릭 시 예측 바 + 근거 요약 펼쳐짐 + "전체 브리핑 보기 →" 링크.

### 변경 후 구조

아코디언 제거, 한 줄 카드 리스트로 교체.

**카드 한 줄:**
```
[ 날짜 ] [ 코스피/미국 뱃지 ] [ 상승우위/하락우위 뱃지 ] [ ✓/✗/- ] [ → ]
```

- 카드 전체가 `<a>` 링크 → 클릭 시 해당 브리핑 페이지 직접 이동
- 아코디언 expand/collapse 없음
- 정확도 결과(`✓`/`✗`): `accuracy.recent_dots`에서 해당 날짜 매핑. 없으면 `-`

## 파일 변경 범위

| 파일 | 변경 내용 |
|------|----------|
| `scripts/templates/index.html` | 아카이브 섹션 마크업 교체 + 정확도 카드 마크업 변경 |
| `scripts/templates/briefing.html` | 정확도 카드 마크업 변경 + 공포탐욕지수 블록 추가 |
| `scripts/generate_html.py` | `streak` 계산 추가, `sidebar_items` 유형별 분기 |
| `web/assets/style.css` | 카드 리스트 스타일 추가, 정확도 카드 Option C 스타일 |
| `web/assets/main.js` | archive용 `toggleAccordion` 호출 제거 (최신 브리핑용은 유지) |

---

## 비변경 항목

- 최신 브리핑(오늘 자) 아코디언 헤더/바디 구조: 유지
- `toggleAccordion` 함수: 유지 (최신 브리핑 접기용)
- 기존 브리핑 flat 파일 (`YYYY-MM-DD-kospi.html` 등): 재생성 불필요 (신규 브리핑부터 적용)
