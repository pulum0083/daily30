# 주간 리포트 에이전트

매주 일요일 21:00 KST에 실행된다. 돌아오는 주(월~금)에 투자자가 반드시 체크해야 할 핵심 이슈를 정리한 주간 리포트를 생성한다.

## Step 1: 데이터 수집

```bash
python3 scripts/fetch_data.py --type weekly
```

웹 검색으로 다음 항목을 직접 조사한다:

### A. 지난 주 시장 리뷰
- 코스피 주간 등락률, 주요 이슈
- S&P500/NASDAQ/다우 주간 등락률
- 주간 섹터별 성과 (상위/하위 3개 섹터)
- 주간 테마주 동향 (AI, 반도체, 2차전지, 바이오 등)

### B. 돌아오는 주 핵심 이벤트 캘린더
수집 대상:
- **미국 경제지표**: CPI, PPI, FOMC, PCE, 고용보고서, GDP, 소매판매 등 (날짜+시각+예상치)
- **연준 위원 발언**: 예정된 연설/발언 일정
- **실적 발표**: S&P500 주요 기업 실적 발표 일정 (특히 빅테크, 반도체)
- **한국 이벤트**: 금통위, 주요 대기업 실적, 공정위/정책 이슈
- **글로벌 이벤트**: G7, OPEC 회의, 주요국 중앙은행 결정
- **IPO/락업해제**: 주요 IPO, 대규모 락업 해제 일정

### C. 핵심 리스크 체크리스트
현재 시장에서 주목해야 할 매크로 리스크:
- 금리/인플레이션 경로
- 지정학적 리스크 (중동, 러우, 미중 갈등)
- 환율 리스크 (원/달러, 엔/달러)
- 국내 수급 리스크 (외국인/기관 동향)

### D. 돌아오는 주 주목 테마 & 종목
- AI/반도체 업사이클 지속 여부 관련 종목
- 실적 시즌 주목 종목 (어닝 서프라이즈 가능성)
- 정책 수혜 테마 (새로 부상하는 이슈)
- 주의해야 할 종목/섹터 (오버밸류, 락업해제 등)

## Step 2: 주간 리포트 생성

### 리포트 구조

```
# 주간 투자 리포트 | [월] ~ [금]

## 📊 지난 주 시장 요약
- 코스피: X,XXX.XX (주간 ±X.X%)
- S&P500: X,XXX.XX (±X.X%)
- NASDAQ: XX,XXX.XX (±X.X%)
- 주간 키워드: [2~3개 핵심 테마]

## 📅 이번 주 이벤트 캘린더
[날짜별 핵심 이벤트 표]

## ⚠️ 이번 주 핵심 리스크
1. [리스크 1]
2. [리스크 2]
3. [리스크 3]

## 🔍 주목 테마 & 종목
[테마별 종목 리스트]

## 💡 이번 주 전략 포인트
[3줄 요약 전략]
```

## Step 3: HTML 주간 리포트 생성

분석 결과를 `data/analysis_weekly.json`에 저장한다.

```bash
python3 scripts/generate_html.py --type weekly --data-file data/latest_weekly.json
```

저장 경로:
- `web/briefings/YYYY-MM-DD-weekly.html`
- `web/index.html` (덮어쓰기)

## Step 4: 텔레그램 메시지 파일 저장

텔레그램 메시지를 `data/telegram_message_weekly.txt`에 저장한다.
(실제 전송은 GitHub Actions workflow가 담당하므로 send_telegram.py를 직접 실행하지 않는다)

```
📋 주간 투자 리포트 | [MM.DD] ~ [MM.DD]

📊 지난 주 요약
• 코스피 [±X.X%] | S&P500 [±X.X%] | NASDAQ [±X.X%]

📅 이번 주 핵심 이벤트
• [날짜] [이벤트명] — [영향도]
• [날짜] [이벤트명] — [영향도]
• [날짜] [이벤트명] — [영향도]

⚠️ 주요 리스크
• [리스크 1]
• [리스크 2]
```
